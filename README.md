# Oasis_Infobyte_Task1
Exploratory Data Analysis (EDA) on Retail Sales Data
🎯 Objective
Analyze the dataset, clean it, explore patterns, and extract meaningful insights using Python.
📝 Steps Performed
- Loaded and explored the dataset
- Cleaned data (handled missing values, removed duplicates)
- Conducted exploratory data analysis (EDA)
- Created visualizations to highlight trends
🛠️ Tools Used
- Python (Google Colab)
- Pandas, NumPy
- Matplotlib, Seaborn
📊 Outcome
A clean dataset with clear insights and visualizations that summarize the main trends and support data understanding.
